# Compendium Features

## Core UI
- Cross - Command center for all actions
- BT View - Office detail workspace with customizable windows
- Offices List - Spreadsheet view of all offices
- Projects List - Spreadsheet view of all projects
- Regulations List - Spreadsheet view of all regulations
- Meditations - Text note storage
- Map View - Barcelona architecture office map

## Data Input
- Manual Entry - Type data directly in spreadsheets
- AI Note Processing - Write natural language, AI extracts data
- Google Places Scraping - Import office data from Google

## AI Features
- AI Orchestrator - Natural language command processing
- Data Extraction - Parse unstructured text into structured data
- Entity Recognition - Identify offices, projects, regulations from text

## Data Management
- Offices Collection - Architecture firms
- Projects Collection - Building projects
- Regulations Collection - Laws and codes
- Workforce Collection - Employee data
- Relationships Collection - Connections between entities
- Meditations Collection - Text notes

## Automation
- Real-time Database Sync - Live updates across all views
- Auto ID Generation - System-generated office IDs
- Connection Counting - Auto-calculate relationships
- Employee Counting - Auto-sum workforce data
- Background Services - Monitor data changes

## Database
- Firebase Firestore - Cloud database
- 31 Collections - 6 active, 25 dormant for future
- Automatic Timestamps - Track creation and updates
- Validation - Schema enforcement

## Window Management
- Dynamic Resizing - Auto-adjust window size per view
- Cross Size - Narrow window for command center
- Spreadsheet Size - Full-width for data tables

## Future Features (Designed, Not Built)
- Market Consolidator - HHI, Gini, CR4/CR8 metrics
- Pulse - Daily automated intelligence collection
- Network Graph Analysis
- 24/7 Background Automation
- Web/Mobile App
